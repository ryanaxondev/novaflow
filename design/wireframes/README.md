# Homepage Wireframes

This directory contains the low-fidelity wireframes for the NovaFlow homepage.

The Markdown specification is the source of truth.

Wireframes are visual representations of the approved specification and are used exclusively for layout and structural review.

## Review Rules

The specification is the source of truth.

Wireframes must never introduce new product decisions.

If a wireframe conflicts with the specification, the specification takes precedence.

Wireframe reviews focus on structure, hierarchy, and user flow—not visual styling.
