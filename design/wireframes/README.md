# Homepage Wireframes

This directory contains the approved low-fidelity wireframes for the NovaFlow homepage.

The wireframe specification is the source of truth. Wireframes are generated from the approved specification and are used to review layout, information hierarchy, and user flow before visual design begins.

---

## Review Rules

- The specification is the source of truth.
- Wireframes visualize approved product decisions.
- Wireframes must not introduce new product decisions.
- If a wireframe conflicts with the specification, the specification takes precedence.
- Reviews focus on structure, information hierarchy, and user flow—not visual styling.

---

## Versioning

Each wireframe version is generated from its matching specification version.

Example:

- `homepage-wireframe-specification-v1.md`
- `homepage-wireframe-v1.png`

Approved versions remain in the repository to preserve the project's design history.
